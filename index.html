How it works

step 1

Pick a scraper

Choose from the catalog and set an input — a keyword, category, or just the defaults.

step 2

Run or schedule it

Run once, or schedule a daily pull on Apify. No servers, no proxies to manage.

step 3

Get clean JSON

Export to JSON, CSV or Excel, or wire the API into a sheet or webhook.

step 4

Or use an AI agent

Every scraper is callable from Claude and MCP-compatible agents via mcp.apify.com — no glue code.

Built to be trusted

Public data, no PII

Only public catalog data — no seller nicknames, reviewer identities, or personal profiles are ever collected.

Honest field lists

Every listing documents exactly which fields populate. Anything a source doesn't expose is disclosed, never advertised.

Pay per result

You pay for rows returned — a few cents per run. Apify's free monthly credit covers your first ~2,000 results.
